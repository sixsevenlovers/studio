# **App Name**: Habitual

## Core Features:

- Habit Creation: Allow users to define new habits, set frequency, and choose time of day.
- AI-Powered Suggestion: Suggest habits based on user-defined goals and interests, with an LLM tool incorporating existing routines from a calendar when constructing new routines.
- Progress Tracking: Visually track habit completion with progress bars and calendars.
- Reminders & Notifications: Send scheduled reminders to complete habits at the set times.
- Streaks & Achievements: Motivate users with streaks, badges, and achievement milestones.

## Style Guidelines:

- Primary color: Deep sky blue (#3399FF), for a calm, trustworthy, and productive feel.
- Background color: Light gray (#F0F5F9), to ensure comfortable contrast in a light color scheme.
- Accent color: A deep shade of violet (#6633FF), for highlighting important actions and milestones; placed to the 'left' of sky blue on the color wheel.
- Body and headline font: 'PT Sans' for a friendly yet contemporary look.
- Use consistent, minimalist icons to represent habit categories and actions.
- Clean and organized layout, prioritizing habit list and progress visualization.
- Smooth transitions and subtle animations for task completion and progress updates.
